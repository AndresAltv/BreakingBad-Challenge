# Built With ViteJS

## Installation

1. Clone the repository<br>
```js
git clone https://github.com/AndresAltv/BreakingBad-Challenge.git
```
<br><br>

2. Install dependencies<br>
npm install<br><br>

3. Start server<br>

npm run dev
